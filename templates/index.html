VyOS Configuration Manager

This is the backend API server for the VyOS Configuration Manager.

If you're seeing this page, it means the backend is running correctly!

You should be automatically redirected to the frontend application.

If you are not redirected automatically, please visit http://localhost:3000.

API endpoints available:

  • GET /api/config - Get the full configuration
  • POST /api/config/interface - Update an interface configuration
  • POST /api/config/save - Save the configuration
  • GET /api/system/status - Get system status information
